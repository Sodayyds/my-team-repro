文章开发.EXE 系统架构文档
1. 整体架构分层
程序采用模块化并行调度架构，分为5层：调度核心层、四大业务功能模块层、资源存储层、输出打包层、工具协作层（Git版本管理）。

2. 四大核心业务模块结构

1. 文章撰写子模块
    职责：文献检索、文稿生成、校对优化；独立运行，可单独更新文献库与写作Prompt。
2. AI绘图子模块
    职责：配图提示词管理、图片生成、图文匹配；输出图片缓存至本地资源目录。
3. 信息配置子模块
    职责：作者、期刊数据读写、投稿压缩包生成；统一管理表格配置文件。
4. 自动投稿子模块
    职责：平台自动化操作、验证码识别、稿件返修处理；独立隔离，避免账号风险影响其他模块。

3. 调度层工作流程
调度核心统一分发任务，四大模块并行执行，全部任务完成后汇总数据，调用输出模块生成完整成品文稿。
