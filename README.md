文章开发.EXE 项目仓库
本仓库用于存储「文章开发.EXE」自动化文稿工具全套开发文档，实现团队Git协同版本管理。

项目简介
文章开发.EXE是一款论文、自媒体文章一站式自动化生成与投稿工具，拆分四大并行业务模块，自动完成文稿撰写、配图、资料打包、多平台投稿，输出万字级带参考文献、插图的完整稿件。

仓库文档说明
- docs/requirements.md：完整项目功能需求文档
- docs/architecture.md：系统模块化架构与团队开发流程

仓库协作方式
1. 使用Issues记录各模块开发、优化任务；
2. 多分支隔离开发：main稳定分支、Dev开发分支、feature-test测试分支；
3. 分支修改完成后创建PR，审核合并至主分支，留存完整迭代记录。
